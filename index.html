dhclient

dhclient est un script que j'ai développé en bash pour macOS durant mon temps libre.

Il reprend le concept de la commande du même nom sous Linux qui permet de gérer les adresses IP dynamiques en utilisant le protocole DHCP. Ce script est utile pour les administrateurs système et les utilisateurs avancés qui ont besoin de configurer et de gérer les paramètres réseau de manière efficace.

Import OVA

proxmox_import_ova est un script que j'ai développé en bash pour Proxmox dans le cadre d'un projet de groupe. L'objectif de ce projet était de déployer et maintenir un serveur Proxmox sur lequel nous devions installer une machine virtuelle.

Il permet d'exporter une machine virtuelle au format OVA sur VirtualBox depuis la machine locale vers un serveur Proxmox ainsi que de l'installer. Le but est que la machine exportée soit directement opérationnelle sur Proxmox.
